🌊 Ekman Transport Visualizer

해양학적 현상을 시각화하는 전문 도구입니다. 에크만 수송 이론을 기반으로 바람과 해류의 상호작용을 분석합니다.

📦 v4.8.1 | 🔧 Python + JavaScript | 📊 Plotly.js

⚡ 주요 기능

  • 일반 바람 조건 시뮬레이션
  • 강풍 조건 분석
  • 태풍 조건 모델링
  • 3D 시각화
  • 2D 분석 차트
  • 결과 내보내기

🔬 기술 정보

  • 에크만 수송 이론
  • 코리올리 효과
  • 에크만 나선
  • 바람 응력 계산
  • 수직 난류 모델링
